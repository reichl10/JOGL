Tries to return requested object. Consults caches first and updates hash\-Last\-Used and entry in {\ttfamily tree} if object found in cache. When found in other cache than primary cache, put object in primary cache. If object not found in caches, consult {\ttfamily Source}, add to pending\-Requests and return null with response\-Type A\-S\-Y\-N\-C\-H\-R\-O\-N\-E. If no Source set, return null and response\-Type M\-I\-S\-S\-I\-N\-G instead.
